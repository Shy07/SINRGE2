关于 SINRGE2
==================

SINRGE2，SINRGE2 Is Not Ruby Game Engine 2 的递归缩写，是由 Sherry Lynch（或称 Shy07）个人开发的一个用于游戏开发的 Ruby 实现。

这是SINRGE2的第二个版本，目前正在开发中。

这个版本放弃了NGE2的代码，转用HGE作为基础。但是考虑到作者对NGE2的函数颇为熟悉，因此这个版本保留了NGE2的部分函数。并进行了适当改进。
