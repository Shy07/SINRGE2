Plane

平面的类。平面是将位图平铺在画面上的特殊精灵。

超类

  • Object

类方法

Plane.new([viewport])

生成一个 Plane 实例。必要时可以指定一个 Viewport。

方法

dispose

释放 Plane。若是已经释放则什么都不做。

disposed?

Plane 已释放时返回 true。

属性

bitmap

Plane 所使用的 Bitmap。

viewport

用于显示 Plane 的 Viewport。

visible

Plane 的可见状态。true表示可见。

z

Plane 的 Z 坐标。数值愈大的平面愈靠近玩家。Z 坐标相同时，后生成的实例会更靠近玩家。

ox

Plane 起始点的 X 坐标。滚动平面时修改此数值。

oy

Plane 起始点的 Y 坐标。滚动平面时修改此数值。

zoom_x

Plane 的横向放大率，1.0 表示原始大小。

zoom_y

Plane 的纵向放大率，1.0 表示原始大小。

opacity

Plane 的不透明度（0～255）。超出范围的数值会自动修正。

blend_type

Plane 的合成方式（0：正常、1：加法、2：减法）。

color

Plane 的合成颜色（Color），alpha 值作为合成的比例。

tone

Plane 的色调（Tone）。
